# Развертывание локальной среды разработки Symfony 7 на Ubuntu 24.04 (Hyper-V)

Документация по конфигурации сетевого окружения, IaC-контейнеров баз данных и решению проблем с биндингом интерфейсов бэкенда ИИ-платформы **MediaHard** [v0.30.0].

---

## 1. Архитектура Dev-контура

Для обеспечения максимальной скорости отладки, прямого доступа к панели `Web Profiler` и минимизации накладных расходов памяти на ВМ Hyper-V, развернута гибридная схема:
*   **Инфраструктурный слой (СУБД PostgreSQL 16 + Mailpit):** Изолированы в Docker Compose.
*   **Слой веб-приложения (Symfony 7):** Запускается локальным движком, но с принудительной сетевой маршрутизацией через конфигурацию окружения.

---

## 2. Пошаговый алгоритм запуска и сетевой Bypass

### Шаг 2.1. Инициализация инфраструктуры Docker
Перед запуском бэкенда поднимаются контейнеры баз данных и почтового перехватчика:
```bash
cd /home/taa/mediahard-app
docker compose up -d
```

### Шаг 2.2. Разрешение блокировки интерфейсов (Bypass)
По умолчанию современные версии Symfony CLI в фоновом режиме (`-d`) игнорируют внешние сетевые интерфейсы и сбрасывают биндинг на локальную петлю `127.0.0.1`, делая сервер недоступным для хост-системы Windows 10/11.

**Решение проблемы:** Жесткое переопределение базового URI и конфигурации СУБД на уровне локальных переменных среды.

Создайте файл `/home/taa/mediahard-app/.env.local` (он автоматически игнорируется нашим `.gitignore`) и внесите следующие параметры:

```text
# Конфигурация подключения к изолированной СУБД Docker
DATABASE_URL="postgresql://app:!ChangeMe!@127.0.0.1:5432/app?serverVersion=15&charset=utf8"

# Принудительный сетевой биндинг Symfony на внешний IP-адрес Hyper-V ВМ
DEFAULT_URI="http://192.168.1.212:8000"
```

### Шаг 2.3. Запуск веб-сервера
После фиксации переменных сред в `.env.local`, запустите сервер стандартной командой:
```bash
symfony server:start -d --allow-http --allow-all-ip
```
*Благодаря `DEFAULT_URI` фреймворк корректно инициализирует маршруты и открывает порт `8000` наружу во внешнюю локальную сеть.*

---

## 3. Картография портов и диагностика окружения

После успешного бутстрапа контур разработки MediaHard доступен по следующим адресам:

*   **Symfony 7 (Бэкенд + Профайлер):** `http://192.168.1.212:8000` — Точка входа в приложение.
*   **Ops Wiki (База знаний MkDocs):** `http://192.168.1.212:8090` — Текущий инженерный портал.
*   **Mailpit (SMTP-интерфейс):** `http://192.168.1.212:32770` — Веб-панель для перехвата писем регистрации.

### Полезные команды отладки:
```bash
# Просмотр логов Symfony в реальном времени
symfony server:log

# Проверка статуса инфраструктурных контейнеров
docker compose ps
```
