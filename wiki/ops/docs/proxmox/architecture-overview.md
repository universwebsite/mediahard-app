# Архитектура платформы MediaHard

## 🏗 Инфраструктура (Infrastructure as Code)
Проект развернут на отказоустойчивом кластере:
- **Гипервизор:** Proxmox (3 узла).
- **Хранилище:** CephFS (обеспечивает репликацию данных и высокую доступность).
- **Контейнеризация:** Docker Compose.

## 🛠 Технологический стек
- **Backend:** PHP / PostgreSQL (надежность и транзакционность).
- **AI-слой:** Ollama + Vision модели (локальный запуск на CPU для анализа стресс-тестов).
- **Мониторинг:** Prometheus + Grafana (контроль состояния HDD и здоровья кластера).

## 🛡 Безопасность и Эксплуатация
- **Цифровой паспорт:** Каждая единица товара привязана к результатам тестов, хранящимся в Ceph.
- **Incident Management:** Ведется журнал инженерных инцидентов (см. `INCIDENT-001.md` в этой папке).
```text
[Internet] -> [Nginx Proxy] -> [Docker: Symfony App]

                                      |
                    [Docker: PostgreSQL] <-> [CephFS Storage]
                                      |
                    [Docker: Ollama AI]  <-> [Media Data]
```
